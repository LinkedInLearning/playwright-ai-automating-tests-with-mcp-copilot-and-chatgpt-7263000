# Glossary (Domain Terms)

Use these terms consistently in code, APIs, and specs.

| Term      | Meaning                                                                    |
| --------- | -------------------------------------------------------------------------- |
| **Bug**   | A single reported issue (title, description, status, etc.) in the tracker. |
| **Board** | The view that lists all bugs (the "board" page).                           |
| **User**  | Someone who logs in and can view/create/edit/delete bugs.                  |

_(Add more terms as the domain grows.)_
