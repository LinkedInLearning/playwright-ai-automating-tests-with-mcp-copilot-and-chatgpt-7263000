# User Story

As a web user,
I want to see a favicon that looks like a classic air-cooled Volkswagen Beetle when I load the BuggyBoard app,
So that I can feel the clean, creative branding of the app.


# Acceptance Criteria

- The BuggyBoard app shows a favicon that looks like a tan-colored 1970 Volkswagen Beetle.
- The favicon's art style is cartoon-y.

# Implementation

- The favicon image is **`frontend/public/favicon.ico`**. The app must reference this file as the site favicon.
