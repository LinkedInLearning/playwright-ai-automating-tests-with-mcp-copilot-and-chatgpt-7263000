# User Story

As a BuggyBoard user,
I want to have my own username and password,
So that I can have my own unique identity with the app.


# Acceptance Criteria

- All user accounts are stored in a file named `users.json` that is located in the project's root directory.
- Each user account has a username and password.
- Every user account must have a unique username.
- There is a default user with username `buggy` and password `1970beetle`.
